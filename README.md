# Habit Tracker

A simple habit tracking app built with vanilla JavaScript.

## Features
- Add, edit, complete, and delete habits
- Bottom sheet UI for habit creation and editing
- Daily completion summary
- Data persistence using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## What I learned
- State-driven UI rendering
- Event handling and DOM manipulation
- Persisting data with localStorage
- Reusing UI components for create/edit flows

## Future Improvements
- Date-based habit tracking
- Weekly calendar view
- Streak tracking
